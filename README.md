# SplixUnlocker 🔓

**SplixUnlocker** — это форк [Bookra1n A5 Afterlife](https://github.com/LinhOnii/BookRa1n), переработанный и улучшенный.  
Инструмент для обхода **iCloud Activation Lock** на устройствах Apple с чипом **A5** (iPhone 4s, iPad 2, iPad mini 1, iPod touch 5).

---

## ✨ Что нового (отличия от оригинала)

| Функция | Оригинал Bookra1n | SplixUnlocker |
|---------|-------------------|---------------|
| **Графический интерфейс** | Есть (PyQt5) | ✅ Улучшен, анимации |
| **Ошибка `coroutine`** | Была на новых версиях `pymobiledevice3` | ✅ **ИСПРАВЛЕНО** — работает на Python 3.11+ |
| **Вес `.exe`** | ~150 МБ | ✅ **ЛЕГКОВЕСНЫЙ** (вырезаны лишние зависимости) |
| **Прогресс-бар** | Нет (только текст) | ✅ Добавлен |
| **Авто-перезагрузка** | Нет | ✅ Добавлена (после `Done!`) |
| **Логи в файл** | Нет | ✅ Сохраняются в `bypass.log` |
| **Батник за 15 секунд** | Нет | ✅ Скоро |

## 📱 Поддерживаемые устройства

| Устройство | Модель | Чип |
|------------|--------|-----|
| iPhone 4s | iPhone4,1 | A5 |
| iPad 2 (все версии) | iPad2,1 – iPad2,7 | A5 |
| iPad mini (1‑го поколения) | iPad2,5 – iPad2,7 | A5 |
| iPad 3 | iPad3,1 – iPad3,3 | A5X |
| iPod touch 5 | iPod5,1 | A5 |

> ⚠️ **Важно:** После обхода **сотовая связь (звонки, 3G/4G) работать не будет**. Только Wi‑Fi.  
> Для iPad Wi‑Fi это не проблема — у них её и не было.

## 📥 Установка и использование
### Способ 1 — Из исходников (Python)

```bash

cd SplixUnlocker

# 2. Установи зависимости
pip install pymobiledevice3 PyQt6

python main.py
